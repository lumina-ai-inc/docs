To get started with Chunkr AI, follow these simple steps to set up your account and integrate our API into your application. We support a [variety of document types](/docs/task-system/file-types). 

<Steps>
    <Step title= "Sign Up and Create an API Key">
        1. Visit [Chunkr AI](https://chunkr.ai)
        2. Click on "Login" and create your account
        3. Once logged in, navigate to "API Keys" in the dashboard
    </Step>
    <Step title = "Install our client SDK">
        <CodeGroup> 
        ```bash Python
        pip install chunkr-ai
        ```
        </CodeGroup>
    </Step>
    <Step title = "Parse a document">
        <CodeGroup> 
            ```python Python
            from chunkr_ai import Chunkr

            # Initialize the Chunkr client with your API key - get this from https://chunkr.ai
            chunkr = Chunkr(api_key="your_api_key")

            # Upload a document via url or local file path
            url = "https://chunkr-web.s3.us-east-1.amazonaws.com/landing_page/input/specs.pdf"
            task = chunkr.upload(url) 
            ```
        </CodeGroup>
    </Step>
    <Step title = "Explore Output">
        <CodeGroup>
        ```python Python
                # Access the chunks from the output
        if task.status == "Succeeded":
            print(f"Document has {len(task.output.chunks)} chunks.")
            # `chunk.content` holds the 1:1 structured representation of document elements
            for chunk in task.output.chunks:
                print(chunk.content)
        else:
            print(f"Task failed with status: {task.status}")
        ```
        </CodeGroup>
    </Step>
    <Step title = "Clean up">
        <CodeGroup> 
            ```python Python
            chunkr.close()
            ```
        </CodeGroup>
    </Step>
</Steps>

## Authentication Options

You can authenticate with the Chunkr AI API in two ways:

1. **Direct API Key** - Pass your API key directly when initializing the client
2. **Environment Variable** - Set `CHUNKR_API_KEY` in your `.env` file

<CodeGroup> 
```python Python
from chunkr_ai import Chunkr

# Option 1: Initialize with API key directly
chunkr = Chunkr(api_key="your_api_key")

# Option 2: Initialize without api_key parameter - will use CHUNKR_API_KEY from environment
chunkr = Chunkr()
```
</CodeGroup>
